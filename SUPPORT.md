# Support

LCE is currently an experimental project and does not provide production SLA,
managed hosting, or compatibility guarantees.

Use public issues for reproducible bugs and feature proposals. For private
security reports and Organization-use requests, use
<https://utakataservice.com/contact/contact.php>. Do not include credentials,
production secrets, or sensitive customer data. Questions should include the
LCE version, Pack/Profile hashes, test command or trace reference, expected
behavior, and actual behavior.
